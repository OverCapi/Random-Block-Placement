# Random Block Placement

Place blocks from your inventory at random or with your own custom probabilities.

Whether you're building organic structures, adding variation to walls, or creating randomized patterns, this mod lets you control exactly how often each selected block is placed.

**Random Block Placement is a client-side mod**, meaning it works on any server without requiring installation on the server or any special permissions. It can be used in both **Survival** and **Creative** mode.

## Features

- 🎲 Random block placement
- ⚖️ Custom weight/probability for every block
- 🎒 Works directly with blocks in your inventory
- 💻 Simple in-game configuration menu
- ⚡ Toggle the mod on or off instantly
- 🌐 Client-side only — no server installation required
- ⛏️ Works in both Survival and Creative mode

---

## How to Use

### 1. Open the selection menu

Press **B** to open the block selection menu.

### 2. Select your blocks

Click on the blocks you want to include in the random selection.

### 3. Set custom probabilities (optional)

Hold **Shift** and click a selected block.

A small input field will appear where you can enter its weight/probability.

Press **Enter** to confirm the value.

### 4. Save your configuration

Click the green **S** button to save your selection.

### 5. Enable Random Placement

Close the menu and press **J** to enable or disable Random Block Placement.

Now simply place blocks as usual!

---

## Reset

Want to start over?

Click the red **R** button in the menu to clear your current selection. Don't forget to save with de Save button

---

## Keybinds

| Key | Action |
|------|--------|
| **B** | Open the configuration menu |
| **J** | Enable / Disable random placement |
